## 欢迎使用mg博客模板
本博客模板网站使用vitepress框架搭建，网站基本配置可以在根目录下的config.yml文件中进行修改,细微的修改大家可以去学习一下vitepress有关的内容，
前提是你已经学会了html,css,js,以及vue

## 安装依赖
```shell
npm i
```

## 项目启动
```shell
npm run dev
```

## 项目打包
```shell
npm run build
```

## 打包文件预览
```shell
npm run preview
```
